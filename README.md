# Таверна на краю миров

Игровой прототип на Unreal Engine 5 в жанре Action RPG с карточной боевой системой в реальном времени.

Проект создаётся в рамках учебного курса по Unreal Engine(OTUS) и постепенно развивается от базового прототипа к полноценной игровой системе.

## Концепция

Игрок исследует Туманный лес, сражается с монстрами, использует карты-заклинания и карты существ, добывает артефакты и возвращается в таверну-хаб для подготовки к следующему заходу.

Основная идея проекта:
- real-time combat
- карточная система
- AI-монстры
- атмосфера dark fantasy
- цикл «таверна → лес → артефакт → возвращение»

## Текущий фокус разработки
